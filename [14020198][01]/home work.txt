Homework week 1
1.Write a C++ program to read a sentence from keyboard and print the
 sentence in to the file “rev.txt” in reversed order. For example: “hello” -> “olleh”.

2.Write a  C++ program including 2 classes: Point and Line. 
Read four points from file “points.txt” and check if two lines are across each other.
 If yes, write out their intersection point.

3.Write a C++ program to read a list of complex numbers from the text file “complex.txt” 
and write out to the screen the sum of the list.

4.Write a C++ program to find the greatest common divisor of two integer numbers. 

5.Write a C++ program to read a list of real numbers from the text file “numbers.txt” 
and output to the file “numbers.out” the list of numbers after increasingly sorted.

6.Given 5 real numbers, write a C++ program to 
calculate the sum of the greatest and smallest numbers.

	Out : ra màn hình;
